安装不同版本的Node/Npm

安装前卸载安装的NodeJS,避免安装后出错

查看本地安装的所有版本；有可选参数available，显示所有可下载的版本。

nvm list [available]

安装，命令中的版本号可自定义，具体参考命令1查询出来的列表

nvm install 14.15.4

使用特定版本

nvm use 14.15.4

卸载

nvm uninstall 14.15.4

常用命令

nvm use 14.15.4

切换到最新版

nvm use node
